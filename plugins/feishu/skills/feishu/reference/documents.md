# Documents

## Search Documents

```yaml
tool: mcp__lark-mcp__docx_builtin_search
data:
  search_key: "产品方案"
  count: 10
useUAT: true
```

## Read Raw Content

```yaml
tool: mcp__lark-mcp__docx_v1_document_rawContent
path:
  document_id: "doxcnxxxxxx"
useUAT: true
```

## Import Document

```yaml
tool: mcp__lark-mcp__docx_builtin_import
data:
  file_name: "report.md"
  content: "# Report\n\nGenerated by Codex."
useUAT: true
```

## Notes

- Prefer user token mode for documents the user should open directly.
- Document URLs often include the document ID after `/docx/`.
- Wiki pages may point to underlying document tokens. Resolve the Wiki node before reading content.
